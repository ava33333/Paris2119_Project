FRCS 102 Paris 2119 Project

Welcome to my Paris 2119 Project website. Throughout the FRCS 102 course, we discussed various different bande desinées (BDs). For our final project, we got to decide what we would create. The website is set up as an interactive demonstration, which slowly develops my thesis, similar to how a French paper would be laid out. The best way to experience this website is by clicking through each page in the order I provide below. You can also check out each page individually by navigating to the menu on the top left of each page.
